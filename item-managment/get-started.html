Item Management: Get Started

Overview of the User interface and introduction to LodDepot

LodDepot offers comprehensive item management solutions, ensuring smooth operations for distributors and sellers. This page will help users to get started with their inventory and start building the catalog. 

Create a New Category

For a more efficient item management, it is important to first create a category for your items. 

  1. Select a parent category if you are creating a sub-category
  2. Manage the visibility of the category by toggling some of the features on or off.
  3. These fields will help you manage the order of your categories within the user suite, the catalog and invoices.
  4. Apply styles to categories to make your catalog easier to navigate.

Create a New Item (for accounts without Quickbooks or other ERP integration)

Now that we have set up our categories. It's time to start populating them with items. Under the Item Management tab, navigate to Items and click Add Item.

  1. Enter UPC or create your personal short item name for easy search and classification. However, you can have both, your personal item name and UPC be displayed at the same time.
  2. Select a base unit in which you would like to track your inventory. When running inventory reports, the QTY on Hand will be displayed in these units.
  3. LodDepot lets you control what units are visible to the customers in catalog and web-stores.
  4. Reorder points help you track your inventory levels. LodDepot will notify you when you are running low.

Create a New Item (for accounts with Quickbooks or other ERP integration)

Now that we have set up our categories. It's time to start populating them with items. Under the Item Management tab, navigate to Items and click Add Item.

  1. Item Name and Description will be synced and managed through your ERP.
  2. Price & Cost will also be synced with your ERP. Make sure that both price and cost correspond to the base unit selected.
  3. Base Unit Select a base unit in which you would like to track your inventory. When running inventory reports, the QTY on Hand will be displayed in these units.
    If a base unit contains more than one item, indicate the amount of items included in the base unit.
    e.g. If your base unit is a Case, which includes 10 items, than enter 10.

Item Units

  1. By setting the price of the base unit, the price of all other units will be recalculated automatically. Cost and price must be related to the base unit.
  2. UNIT 1: Case By leaving the Price field, for our unit 1, empty, price of the case automatically got calculated based on pounds/case.
  3. If a base unit contains more than one item, indicate the amount of items included in the base unit.
    e.g. If your base unit is a Case, which includes 10 items, than enter 10.
  4. UNIT 2: Pallet  In this unit we opted to give our customers a discount of 10% for purchasing this unit. 
    We could also set a fixed price for this particular unit, by unchecking the % box and entering the price.
  5. When your unit is represented by variable weight, then you indicate this by selecting this checkbox.

Item Options

  1. Item Tags will appear in the item catalog and allow users to search by tags as well.
  2. Go to Tools -> Settings -> Tables to create new Custom Tags
  3. Allows Sales Reps to change the price, within the set parameters.
  4. If you are running promotions such as 10+1, checking this box allows sales reps to select the amount of free items.
  5. Non-Catalog Items do not show up in catalog. Generally used for fees, deposits and other special non-catalog items.

Item Brands

Creating Item Brands is an easy way to organize your products. Just create a new brand and LodDepot will automatically find the brand items and assign to corresponding brands. 

  1. LodDepot will automatically assign products into categories based on keywords.
  2. Check this box to sort items into categories only once and opt-out of the automatic brand assignment.
