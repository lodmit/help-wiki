User Management

Creating a New User 

  1. If you are using QuickBooks or another ERP along with LodDepot Reference ID will be used to match the user between the two systems.
  2. Check this box if you want LodDepot to send notifications about each new order that is created.
  3. Administrator
    Has full access to the account. This is the only type of user that can make changes to Company Profile and Security Settings
  4. Manager
    Has the same rights as the Administrator except for Company Profile and Security Settings
  5. Salesman 
    Choosing between Mobile/Office/Truck Salesman only helps to categorize the users more efficiently.
  6. Salesman rights are controlled by using the Options checkboxes

Options

Options are used to control the rights of Salesmen.

  1. The salesman will be able to edit all orders including the orders of other salesmen. 
  2. Set new price or cost, delete, create items.
  3. Edit customers information incl. terms, credit limit add or delete stored files.
  4. By checking this box, the user will only be able to view the customers that are assigned to this sales rep.
  5. Manage price levels.
  6. Manage item lots and inventory.
  7. Cost and profit information will not be visible to this user
  8. The salesman will not be able to edit prices when creating or editing an order

Other Settings

  1. By assigning an Access Group, you can control the visibility of certain categories by this user.
    If you want to allow the user to view a protected category + all other categories, then check this box.
    If you want the user to have access only to categories marked visible only to a certain access group, then leave the box unchecked
  2. If you assign a user to a certain facility than this customer’s view will be limited only by the inventory at that facility.

 

Mobile Access QR Code

To quickly set up a new user's device, scan a QR code that can be generated by selecting the user and clicking More.
